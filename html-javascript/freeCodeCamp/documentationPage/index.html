Introduction

Python is one of the most common programming language for beginners because of its straight forward syntax that's easy to understand for people without programming experience.

What is included in this guide:

  • How to print hello world
  • Variables

Hello World

To have Hello World printed in the console, we will need to do:

> print("Hello World")

Variables

Variables are just like maths where we use x and y to store different values.
Here are a list of different types of variables in python:

  • String
  • Integer
  • Boolean
  • Float

There are more variable types in python but these are all the ones you need to know for now.

To store a variable to given word, we can do:

> name = "John"
> x = 5
> pass = True
> pi = 3.14

Operators

To do calculations in python, we will have to use operator signs as follow:

  • + (plus)
  • - (minus)
  • * (times)
  • / (divide)

Here are some examples of how to use these operators:

> 9 + 10
> 21 - 5
> 6 * 9
> 22 / 7

If else

When dealing with logics, the most used logic command in any programming language is the if else statement
Here is an example of how to use an if else statement:

john = "male"
y = 6

if(john == "male"):
  print("John is a male")

else:
  print("John is not a male")

If there are multiple conditions in a logic, we will have to use else if. In python, else if is combined into a single word elif. Here is an example:

x = 5
y = 6

if(x == y):
  print("x is equal to y")

elif(x < y):
  print("x is small than y")

elif(x > y):
  print("x is bigger than y")
